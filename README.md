# IP Finder (Java Swing)

## 📌 Description

A simple Java Swing application to find the IP address of a given website.

## 🚀 Features

* Enter URL
* Get IP address instantly
* Handles invalid inputs

## 🛠️ Tech Stack

* Java
* Swing
* InetAddress

## ▶️ How to Run

1. Open in NetBeans
2. Run `IPFinderApp.java`
3. Enter domain (e.g., google.com)
